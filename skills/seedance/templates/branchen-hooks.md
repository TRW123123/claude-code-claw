## 1. UNIVERSELLE HOOK-PATTERNS (0-2s) — 27 destilliert

Klassifiziert nach psychologischem Trigger. Jede Zeile = direkt in Seedance-Prompt einsetzbar.

### Pattern-Interrupt
- **Impossible Scale** — Objekt unerwartet groß/klein (Burger so groß wie Auto, Person so klein wie Tasse)
- **Impossible Physics** — Schwerkraft umgedreht, Flüssigkeit fließt aufwärts, Glas zerbricht rückwärts
- **Reverse Motion** — Sequenz läuft sichtbar rückwärts, dann Switch
- **Glitch Effect** — Kurzes Pixel-Flackern (0.3s) → snap clean
- **Audio Jump** — Stille bricht ab mit hartem Bass-Hit auf Visual

### Curiosity Gap
- **Half-Reveal** — Nur Teilausschnitt sichtbar, Rest verdeckt
- **Wait For It Setup** — Frame friert auf "fast"-Moment, Text "wait for it..."
- **Misdirection** — Erwartung A aufbauen, in 2s zu B switchen
- **Freeze Frame Break** — Plötzlicher Hard-Stop mitten in Bewegung
- **Negative Space Break** — Leerer Frame → unerwartetes Element schießt ins Zentrum

### Emotional Trigger
- **Cute Overload** — Tier/Baby-Moment in 1s, dann Produkt-Pivot
- **Fear/Tension** — Dunkles Setup, plötzliches Licht
- **Awe/Wonder** — Cinematischer Weitwinkel-Reveal (Landschaft, Skyline, Interior)
- **Humor/Absurdität** — Unerwartete Komik (z.B. Friseur schneidet vs. Person flieht)

### Contrast
- **Before/After Tease** — 1s Problem-State, 1s Lösung
- **Split Screen Transformation** — Linke Hälfte alt, rechte neu, Wipe-Transition
- **Color Explosion** — Monochrom → Farbe schießt rein

### Movement Detection
- **Extreme Close-up Zoom** — Macro-Detail → Pull-Back zu Full-Shot
- **Camera Look** — Person dreht sich, schaut direkt in Kamera
- **Focus Shift** — Vordergrund unscharf → Hintergrund unscharf
- **Rapid Cuts** — 4-6 schnelle Schnitte in 2s (Marken-Branding-Burst)

### Direkt-Trigger
- **Direct Question** — Text-Overlay "Müde von X?"
- **On-Screen Text Command** — "STOP. Schau hin." als Erst-Frame
- **Ladder Reveal** — 3 Elemente nacheinander einblenden (3× je 0.5s)
- **Inspiration/Aspiration** — Aspirational Lifestyle in 1s + Slogan
- **Shock/Disgust** — Negativer State (verdreckter Spiegel etc.) → Cleanup-Promise
- **Unexpected Reveal** — Etwas Banales öffnet, Premium-Produkt drin

## 2. BRANCHEN-MAPPING — welcher Hook für welchen Use-Case

Auf deine aktiven Projekte gemapped:

### Profilfoto-KI (profilfoto-ki.de)
**Pflicht-Hooks:** Before/After Tease, Split Screen Transformation, Camera Look
- "Selfie verpixelt → Schnipps → Pro-Profilfoto"
- "Person blickt direkt in Kamera mit beidem Output nebeneinander"
- "Bewerbung wird abgelehnt-Stempel → Foto-Switch → angenommen"

### ki-automatisieren.de + yapayzekapratik.com (B2B SMB)
**Pflicht-Hooks:** Direct Question, Before/After, Ladder Reveal
- "Inbox 247 ungelesen → Automation läuft → 0"
- "3 Schritte zur ersten KI-Automation (1s pro Schritt)"
- "Müde von manuellen Mails an Kunden?"

### st-automatisierung.de (BAFA Beratung)
**Pflicht-Hooks:** Awe/Wonder, Inspiration/Aspiration, Direct Question
- Slower-Paced, Quiet Authority — kein Bro-Marketing
- "50% Förderung. 100% umgesetzt." mit cinematic B-Roll
- KEINE Bro-Hooks (Misdirection/Cute/Absurdität)

### Autohaus-Video (DACH+TR Dealer)
**Pflicht-Hooks:** Camera Look, Impossible Scale, Awe/Wonder
- Auto-Detail-Macro → Pull-Back zu Full-Shot mit Verkäufer
- Dealer-Manifest "Wir verkaufen keine Autos, wir...": Inspiration

### Friseur-pSEO + Local-SMB
**Pflicht-Hooks:** Before/After Tease, Cute Overload (für Kinder-Friseur), Color Explosion
- Wash → Cut → Style in 3 Cuts (Ladder)
- Direct Question "Schon mal beim Friseur 2h gewartet?"

### AI-UGC (Instagram Reels für profilfoto-ki / ki-automatisieren)
**Pflicht-Hooks:** Camera Look, Wait For It, Pattern-Interrupt
- Maximaler Hook-Druck — TikTok-Algorithmus straft schwache erste 0.5s

## 3. MONEY-SHOT-LIBRARY — die Kamera-Momente die verkaufen

15 konkrete Setups, branchenagnostisch nutzbar in Prompts:

| Money Shot | Setup (für Seedance-Prompt) |
|---|---|
| **Cheese Pull** | Backlit, side-angle, slow-mo, max stretch vor break |
| **Slow Sauce Pour** | 45° angle, golden backlight, pool-and-shine |
| **Steam Backlit Glow** | Backlight gegen Steam, warm 3000K, macro wisps |
| **Texture Macro Pull-Back** | Start macro auf Texture, smooth zoom-out 2-3s |
| **Hand Interaction Tap** | Floating Device, Hand entry top-right, decisive tap, ripple-response |
| **Product Drop + Bounce** | Dark BG, motion-blur descent, soft glow on landing |
| **Cross-Section Reveal** | Smooth knife/cut, interior in sharp focus, slow push-in |
| **Color Cascade** | 5-12 Varianten rotieren in 2s (Lipstick/Auto-Farben/Profilfoto-Stile) |
| **Splash Catch** | High-speed droplets mid-air, vibrant fluid |
| **Light Beam Scan** | Bright animated beam sweeps across dark interface/face/product |
| **Particle Burst from Logo** | Logo shatters → particles re-form als UI/Produkt |
| **First-Bite/Sip/Touch** | Hand bringt zum Mund, Moment vor Kontakt, Befriedigung |
| **360° Isometric Spin** | Smooth 3D-rotation, gleichbleibender Winkel, 2-3 full rotations |
| **Glitch-to-Clean** | 0.3-0.6s flicker → snap into pristine state |
| **Before/After Wipe** | Vertical/horizontal wipe-line trennt zwei States |

**Empfehlung für Profilfoto-KI:** Color Cascade (verschiedene Stile durchblitzen) + Before/After Wipe + Light Beam Scan (auf Gesicht).
**Empfehlung für Autohaus:** 360° Isometric Spin + Texture Macro Pull-Back (Karosserie-Detail → Full-Car).

## 5. ANTI-PATTERN AUS DEM HIGGSFIELD-REPO

Was du NICHT übernehmen sollst (auch wenn die Templates es vorschlagen):

- ❌ "Influencer hand stop gesture + text overlay 'Wait, this is life-changing'" — Bro-Marketing, verletzt SOUL.md
- ❌ TikTok-Trending-Audio als Pflicht-Element — du willst Voiceover-getriebenes UGC mit SEO-Wert
- ❌ "CTA-Button mit pulsing glow" auf Reels — wirkt Spam-Ad-haft auf DACH-Algorithmus
- ❌ Englische Hook-Texte 1:1 ins Deutsche übersetzen ("Stop Scrolling" → "Hör auf zu scrollen" liest sich grottig)
- ❌ "Limited Drop / FOMO-Hook" für Beratung/B2B — passt nicht zu Quiet Authority
- ❌ Übermäßige Saturation in Profilfoto-Demos — wirkt KI-Slop statt Pro-Foto
