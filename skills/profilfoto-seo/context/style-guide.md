# Style Guide — profilfoto-ki.de

## Anti-AI Blacklist (Hard Rule)

Diese Wörter/Phrasen NIEMALS verwenden:

**Marketing-Floskeln:**
- revolutionär, bahnbrechend, innovativ, wegweisend
- nahtlos, mühelos, spielerisch, intuitiv
- maßgeschneidert, zugeschnitten (selten ok), perfekt auf dich abgestimmt
- modernste, neuartig, zukunftsweisend
- Rekordzeit, blitzschnell, im Handumdrehen

**Hohle Versprechen:**
- nachhaltig transformieren, Leben verändern
- unbegrenzt, grenzenlos, vollständig
- 100% zufrieden, garantiert, ohne Kompromisse

**B2B-Jargon (falsch für B2C):**
- Workflow, Prozess, Lösung, Tool-Stack
- Stakeholder, Zielgruppe, Buyer Persona
- skalierbar, strategisch, holistisch
- Entdecke, Erlebe, Revolutioniere (als Imperative)

**KI-Tech-Jargon:**
- Diffusion Model, Latent Space, Neural Network
- Machine Learning, Deep Learning, Neuronales Netz
- Prompt Engineering, Fine-Tuning

**Ausrufe-Register:**
- Jetzt! Sofort! Nur heute!
- Ausrufezeichen-Ketten (max 1 pro Absatz, oft 0)

## Typografie

| Regel | Richtig | Falsch |
|---|---|---|
| Gedankenstrich | Spiegelstrich (—) oder Halbgeviert (–) | ASCII-Hyphen (-) als Gedankenstrich |
| Aufzählungen | Bullet Points (•) oder HTML-Listen | Gedankenstriche als Listenpunkte |
| Anführungszeichen | „deutsch" oder "auch ok" | 'englisch' |
| Prozent | 80 % (mit Leerzeichen) oder 80% (beides ok, konsistent) | |
| Zahlen < 13 | ausgeschrieben (fünf, zehn) außer in Tabellen/Daten | |
| Zahlen ≥ 13 | Ziffern | |

## Satz-Struktur

1. **Subjekt + Verb früh.** Nicht "In der heutigen schnelllebigen Zeit ist…" sondern "Dein Profilbild entscheidet…"
2. **Aktiv statt Passiv.** "Die KI erstellt dein Bild" statt "Dein Bild wird von der KI erstellt."
3. **Max 25 Wörter pro Satz** als Richtschnur. Länger nur wenn der Gedanke es trägt.
4. **Variable Satzlänge.** Kurz. Mittel. Ab und zu ein längerer, der mehr trägt.
5. **Kein Schachtel-Satz mit >2 Nebensätzen.**

## Absatz-Struktur

- **Max 5 Sätze pro Absatz.** Web-Lesbarkeit.
- **Erster Satz trägt den Gedanken.** Keine Ein-Satz-Einleitungen.
- **Whitespace nutzen** — lieber mehr Absätze als ein Block.

## Überschriften

- **H1:** Einmal pro Seite, enthält Haupt-Keyword.
- **H2:** Gliederungs-Ebene, soll beim Scannen verständlich sein ("Was kostet ein KI-Bewerbungsfoto?" nicht "Die Kostenfrage").
- **H3:** Unter-Sektionen.
- **Niemals H4+** außer in tiefen Ratgebern (>2000 Wörter).

## Tonalitäts-Register (Erinnerung aus brand-voice.md)

- **Du-Form** immer
- **Empathisch** aber nicht kriecherisch
- **Konkret** statt abstrakt
- **Zahlen mit Quelle** ("Princeton-Studie" statt "Studien zeigen")

## SEO-Gerechtes Schreiben

| Regel | Beispiel |
|---|---|
| Keyword in H1, erstem Absatz, URL, Title | "KI Bewerbungsfoto" durchgängig |
| Semantisches Feld abdecken | Synonyme: Bewerbungsfoto, Profilbild, Porträt, CV-Foto |
| Long-Tail als Unterüberschrift | "Wie viel kostet ein KI-Bewerbungsfoto?" (Frage = Featured Snippet Chance) |
| Kein Keyword-Stuffing | Max 1% Dichte, lieber semantisch reich |
| FAQ-Block | Mindestens 4 Fragen am Ende jeder Ratgeber-Seite (für FAQPage Schema) |

## Trailing-Slash Regel (Hard Rule)

Alle internen Links und Canonical-URLs MÜSSEN mit `/` enden:
- ✅ `/bewerbungsfoto-ki/`
- ❌ `/bewerbungsfoto-ki`

## Meta-Längen (Hard Rule)

- **Title:** max 60 Zeichen (wir zählen vor dem Write)
- **Meta Description:** max 155 Zeichen
- **og:title:** identisch mit Title
- **twitter:title:** identisch mit Title

## Quellen-Pflicht (60%-Regel)

Mindestens 60% der nicht-trivialen Aussagen brauchen eine Quelle:
- Statistiken → verlinkte Quelle
- Preise → interner Link auf /preise/
- Behauptungen → externe Referenz oder interne Daten

## Kein Smalltalk im Content

- Keine "In diesem Artikel lernst du…"-Einleitungen
- Keine "Fazit"-Überschrift mit leerem "Zusammenfassend…"
- Keine "Wir hoffen, dass…" Schlussformeln
- Direkt rein, direkt raus.
