## DIE GROSSE REGEL

**Em-Dash:** Wenn du mehr als EINEN Em-Dash (—) pro Seite findest, ueberarbeite den Text. Em-Dash ist die Nummer 1 KI-Erkennung.

**Ersetzungs-Logik fuer Em-Dash (kontextsensitiv):**
- Bei Attribution / Erlaeuterung → Komma
- Zwischen unabhaengigen Saetzen → Semikolon oder Punkt
- Bei abrupter Pause → Klammer oder Punkt
- Vor "however", "therefore", "moreover" → Semikolon
- Bei Großbuchstaben nach dem Em-Dash → Punkt + neuer Satz

## DEUTSCHE BRO-MARKETING / BUZZWORDS (zusaetzlich verboten)

Diese Begriffe sind in deutschem B2B-Beratungs-Content sofortige Disqualifikation:

- "Game-Changer" / "Spielveraenderer"
- "Next Level"
- "auf das naechste Level heben"
- "Disruption" (außer in akademischem Kontext mit Quelle)
- "Skalierung" (wenn nicht technisch praezise)
- "10x"
- "Boost" / "boosten"
- "Hack" / "Lifehack"
- "ultimative Anleitung" / "ultimativer Guide"
- "geheim" / "Geheimwaffe"
- "endlich" (als Marketing-Trigger)
- "Insider-Tipp"
- "Hidden Gem"
- "must-have"
- "Powerhouse"
- "Rocket Science"

## CHECKLIST VOR JEDEM CONTENT-RELEASE

- [ ] **Em-Dash Zaehlung** — max 1 pro Seite, sonst ersetzen
- [ ] **Verbotene Verben** — keine im Text
- [ ] **Verbotene Adjektive** — keine im Text
- [ ] **Verbotene Uebergaenge** — keine im Text
- [ ] **Opening Phrases** — der erste Satz darf KEINE der Listed-Phrases enthalten
- [ ] **Conclusions** — der letzte Absatz darf KEINE Listed-Conclusion enthalten
- [ ] **Filler Words** — minimum 80% der gelisteten Wörter entfernt
- [ ] **Strukturelle Patterns** — keine "Whether you're a..." oder "It's not just..." Konstrukte
- [ ] **Bro-Marketing** — keine Buzzwords aus der DE-Liste
- [ ] **Academic Tells** — keine im Text

**Wenn ein Punkt fehlschlaegt:** Content geht NICHT live bevor er ueberarbeitet wurde.

## UEBERPRUEFUNG VIA REGEX (fuer SQL/Python)

Konzept fuer eine spaetere Quality-Check-Funktion in Supabase:

```sql
-- Einfache Pattern-Detection in claw.changelog new_value
-- Liefert Anzahl Treffer fuer KI-Phrasen
SELECT
  page_path,
  array_length(regexp_split_to_array(new_value, 'nahtlos|revolutionaer|massgeschneidert|entfesseln|im heutigen digitalen Zeitalter|abschliessend lasst sich sagen|davon abgesehen|im Wesentlichen|—'), 1) - 1 AS ai_phrase_hits
FROM claw.changelog
WHERE domain = 'st-automatisierung.de' AND change_type IN ('content', 'meta_desc', 'title')
ORDER BY ai_phrase_hits DESC NULLS LAST;
```
